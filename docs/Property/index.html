  • Property

    • Microdata that describes a property in a schema.

    The schema can be validated with the JavaScript module at https://schema.host/Property/index.mjs. 

        import { validator } from 'https://schema.host/Property/index.mjs';
    if ( await validator( anElement ) ) {
        console.log('Element is valid according to the Property schema.');
    } else {
        console.error('Element is invalid according to the Property schema.');
    }
