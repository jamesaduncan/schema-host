  • Credential

    • A schema for a user's credentials.

    The schema can be validated with the JavaScript module at https://schema.host/Credential/index.mjs. 

        import { validator } from 'https://schema.host/Credential/index.mjs';
    if ( await validator( anElement ) ) {
        console.log('Element is valid according to the Credential schema.');
    } else {
        console.error('Element is invalid according to the Credential schema.');
    }

    Properties of the Credential schema
    Property Name Type Cardinality Description
    username Text 1 The user's login name, or a unique identifier for this credential
    password Text 1 The user's password, encrypted using the method specified in the 'encryption' property.
    role Text 0..n A the role of the user, such as 'administator', 'editor' etc.
